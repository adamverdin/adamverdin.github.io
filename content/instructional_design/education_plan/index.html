Course Title:                Using the dbMotion SDK Toolkit

Course Description:    During this instructor-led course, developers will get hands-on experience
using the dbMotion SDK Toolkit to transfer patient data using Native and Web Services.

Course Duration:         4 days
Offering Frequency:  At least 1 time annually

Laboratory Plan

In order for Performance Exercises to be conducted during the ILT sessions, a laboratory environment will be required. 

·      The environment will require a working dbMotion instance within existing Education Services training network.

·      Allscripts IT will prepare the training environment.

·      dbMotion Development will be consulted to help get the dbMotion instance set up appropriately
