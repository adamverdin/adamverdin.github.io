User Functions

1.    To get user details by UserId

Method

GET

 To get user details by UserId

URL

http://erxilearn.eduserv.myallscripts.com/api/index.php/users/get_by_id/

Authentication

HTTP Basic

 

Header

username: erxrestuser

password: demo_password

 

 

 

 

Data OR

{“UserId”:1}

 

Query Parameters

?UserId=1

 

 

 

 

Return data types

json, xml

 

Return Status

 

 

OK (200)

{
  "UserId": 1,
  "UserName": "JDoe",
  "ClientId":10092012E-F,
  "FirstName": "John",
  "LastName": "Doe"
}

NOT FOUND (404)

{

 "status": FALSE,

 "message": “User not found”

}

BAD REQUEST (400)
